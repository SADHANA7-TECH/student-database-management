# Student Database Management

## Overview

This project is a beginner-friendly SQL mini project developed using MySQL to manage and analyze student records. It demonstrates how databases can be created, structured, and queried to retrieve meaningful insights from student data.

## Objective

The objective of this project is to store student information in a structured database and perform various SQL operations to analyze academic performance and attendance.

## Tools Used

* MySQL
* SQL

## Features

* Create and manage a student database
* Insert student records
* Retrieve and filter data
* Sort records based on marks
* Calculate average, highest, and lowest marks

## Files

* schema.sql — Database and table creation
* data.sql — Sample student records
* queries.sql — SQL queries for analysis

## Sample Queries

* SELECT * FROM students;
* SELECT AVG(marks) FROM students;
* SELECT MAX(marks) FROM students;

## Learning Outcome

This project improved my understanding of database management, SQL query execution, and data analysis using structured datasets.
